Detailed Description

Contains functions for opening a game's configuration and metadata.

Functions are related to a games configuration, directory set, system files, main script name and the name of the game function in JavaScript and game.sgm file. Contains the structures TS_Config and TS_Directories, which are used by plugins to get configuration, system file, and directory information for the current game.
