Detailed Description

Contains functions for opening a game's configuration and metadata. Functions are related to a games configuration, directory set, system files, main script name and the name of the game function in JavaScript and game.sgm file.

Macro Definition Documentation

#define CONFIGMGR_EXPORT   extern "C"

Defines the shared library export code for the platform when CONFIGMGR_INTERNAL is undefined, and the shared library import code for the platform otherwise.
