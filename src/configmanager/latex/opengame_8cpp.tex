\subsection{Detailed Description}
Contains functions for opening a game's configuration and metadata. Functions are related to a games configuration, directory set, system files, main script name and the name of the game function in Java\-Script and game.\-sgm file. Contains the structures \hyperlink{structTS__Config}{T\-S\-\_\-\-Config} and \hyperlink{structTS__Directories}{T\-S\-\_\-\-Directories}, which are used by plugins to get configuration, system file, and directory information for the current game.
